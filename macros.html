Hello Plunker!

мои макросы:

1) сайт (если выделенная ячейка находится на "Лист 1", то макрос вставляет записанный текст "lalalalala" в выделенную ячейку)

Sub сайт()
'
' сайт Макрос
'
' Сочетание клавиш: Ctrl+й
'
If Worksheets(ActiveSheet.Name).Name = "Лист1"
ActiveCell.FormulaR1C1 = "lalalalala"
End If

2) макрос 3 (меняет "lalalalala" на "123")

Sub Макрос3()
'
' Макрос3 Макрос
'
' Сочетание клавиш: Ctrl+л
'
Cells.Replace What:="lalalalala", Replacement:="123",
LookAt:=xlPart, _
SearchOrder:=xlByRows, MatchCase:=False,
SearchFormat:=False, _
ReplaceFormat:=False
End Sub

3) макрос 4 (оформляет ячейку курсивом и жирным шрифтом)

Sub Макрос4()
'
' Макрос4 Макрос
'
' Сочетание клавиш: Ctrl+о
'
Selection.Font.Bold = True
Selection.Font.Italic = True
End Sub
